2. Description of Service

SaveIt is a full-featured private browser for Android that includes:

  • Browser: Multi-tab web browsing with bookmarks, history, search, and the ability to be set as your default browser
  • Media Detection: Detects and downloads video and audio resources from web pages
  • Local VPN Media Detection: Uses Android VpnService locally to detect media links from apps you select (local only, no external routing)
  • Safebox: Protected hidden local file vault accessible after PIN and/or biometric authentication
  • Form Autofill: Detects login forms and saves/fills account credentials locally
  • Ad Blocking: Blocks advertisements and redirects while browsing
  • Download Manager: Manages file downloads including HLS and DASH streams
  • DLNA Cast: Casts media to DLNA-compatible devices on your local network
  • Screen Recording: Records on-screen content with optional audio capture using Android MediaProjection
  • QR Code Scanner: Scans QR codes and barcodes using the device camera
  • Audio/Video Player: Built-in media player with background audio playback support
  • File Import: Receives files shared from other apps for import into Safebox or playback
  • Floating Window: Overlay controls for quick access to media detection while using other apps
  • APK File Handling: Saves, opens, or shares downloaded APK files through Android system flows without directly installing them
  • WiFi Transfer: Sends and receives files between devices on the same local network via a temporary local HTTP server
  • Clipboard Detection: Monitors clipboard for URLs to enable quick navigation (optional, can be disabled)
  • Webpage Screenshot: Captures screenshots of the current webpage or video player content
  • Text Selection Search: Allows searching text selected in other apps within the browser
